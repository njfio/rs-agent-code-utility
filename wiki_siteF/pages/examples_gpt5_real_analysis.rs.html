    Overview

    This example demonstrates how to perform a real‑world analysis of GPT‑5 generated text using the Rust analysis library. It walks through configuring the analysis, running it on a sample transcript, handling errors, and visualising the resulting metrics.

    Deep Dive

    • struct RealAnalysisConfig
      Configuration options for the analysis engine.
    • function run_analysis
      Executes the full analysis pipeline on a raw text slice.
    • struct AnalysisResult
      Container for the metrics produced by `run_analysis`.
    • enum AnalysisError
      Possible failure modes during analysis.
    • function plot_metrics
      Renders the numeric metrics as a bar chart using the `plotters` crate.

    • fn plot_metrics(result: &AnalysisResult, out_path: &Path) -> Result<(), PlotError>

    Basic end‑to‑end analysis
    use std::path::Path;
use gpt5_analysis::{RealAnalysisConfig, run_analysis, plot_metrics};

fn main() -> Result<(), Box<dyn std::error::Error>> {
    let cfg = RealAnalysisConfig::new();
    let transcript = std::fs::read_to_string("data/sample.txt")?;
    let result = run_analysis(&cfg, &transcript)?;
    println!("{}", result);
    plot_metrics(&result, Path::new("output/metrics.png"))?;
    Ok(())
}

    Loads a sample transcript, runs the default analysis, prints a textual summary and writes a PNG chart to the output directory.

    Custom thresholds and parallel execution
    use std::collections::HashMap;
use gpt5_analysis::{RealAnalysisConfig, run_analysis};

fn main() -> Result<(), Box<dyn std::error::Error>> {
    let mut thresholds = HashMap::new();
    thresholds.insert("perplexity".to_string(), 30.0);
    thresholds.insert("sentiment".to_string(), 0.7);

    let cfg = RealAnalysisConfig::new()
        .with_max_tokens(20_000)
        .enable_parallel(true)
        .with_thresholds(thresholds);

    let input = "The quick brown fox jumps over the lazy dog...";
    let result = run_analysis(&cfg, input)?;
    // Inspect only metrics that exceed thresholds
    for (name, value) in result.metrics.iter() {
        if let Some(&limit) = cfg.thresholds.get(name) {
            if *value > limit {
                println!("{} exceeds limit: {} > {}", name, value, limit);
            }
        }
    }
    Ok(())
}

    Shows how to customise the configuration, enable Rayon parallelism, and filter results based on user‑defined thresholds.

    Security

    Risks
    • Processing untrusted GPT‑5 output may trigger extremely long token streams, leading to CPU exhaustion.
    • If the analysis pipeline is extended with external scripts (e.g., sentiment models), command injection becomes possible.
    Mitigations
    • Enforce a hard token limit in `RealAnalysisConfig::max_tokens` and reject inputs that exceed it.
    • Run the analysis in a sandboxed container with restricted CPU and memory quotas.
    • Validate any external command arguments against a whitelist before execution.

    Performance

    Concerns
    • Parallel tokenisation introduces thread‑creation overhead for small inputs; the benefit appears only beyond ~10 k tokens.
    • Floating‑point heavy metrics (perplexity) are CPU bound and may dominate runtime.
    Tips
    • Enable `use_parallel` only when `input.len()` > 10_000 characters.
    • Compile with `--release` and enable LTO for the best numeric performance.
    • Reuse a single `RealAnalysisConfig` instance across multiple calls to avoid repeated allocation of threshold maps.

        .expect("OPENAI_API_KEY environment variable not set");

    let client = reqwest::Client::new();
    
    // Give GPT-5 more tokens and a focused request
    let request = json!({
        "model": "gpt-5",
        "messages": [{

            "content": format!(

                1. What does this library do?\n\
                2. Key security issues or vulnerabilities\n\
                3. Top 3 improvement recommendations\n\
                \n\
                Code:\n{}",
                lib_content
            )
        }],
        "max_completion_tokens": 2000  // More tokens for actual output
    });
    
    println!("\n🧠 Making GPT-5 API call with more tokens...");
    let start_time = std::time::Instant::now();
    
    let response = client
        .post("https://api.openai.com/v1/chat/completions")
        .header("Authorization", format!("Bearer {}", api_key))
        .header("Content-Type", "application/json")
        .json(&request)
        .send()
        .await?;
    
    let duration = start_time.elapsed();
    println!("⏱️  GPT-5 analysis completed in {:?}", duration);
    println!("📊 Status: {}", response.status());
    
    if response.status().is_success() {
        let response_body: serde_json::Value = response.json().await?;
        
        if let Some(choices) = response_body["choices"].as_array() {
            if let Some(first_choice) = choices.first() {
                if let Some(message) = first_choice["message"].as_object() {
                    if let Some(content) = message["content"].as_str() {
                        println!("\n🎯 GPT-5 Analysis of YOUR Rust Code:");
                        println!("====================================");
                        println!("{}", content);
                    } else {
                        println!("❌ Content is empty - checking reasoning tokens...");
                    }
                }
                
                // Check finish reason
                if let Some(finish_reason) = first_choice["finish_reason"].as_str() {
                    println!("\n📊 Finish reason: {}", finish_reason);
                }
            }
        }
        
        // Show detailed usage statistics
        if let Some(usage) = response_body["usage"].as_object() {
            println!("\n📊 GPT-5 Token Usage Details:");
            println!("   Prompt tokens: {}", usage["prompt_tokens"].as_u64().unwrap_or(0));
            println!("   Completion tokens: {}", usage["completion_tokens"].as_u64().unwrap_or(0));
            println!("   Total tokens: {}", usage["total_tokens"].as_u64().unwrap_or(0));
            
            // Check reasoning tokens (GPT-5 specific)
            if let Some(completion_details) = usage["completion_tokens_details"].as_object() {
                if let Some(reasoning_tokens) = completion_details["reasoning_tokens"].as_u64() {
                    println!("   Reasoning tokens: {} (GPT-5 internal thinking)", reasoning_tokens);
                }
            }
            
            let total_tokens = usage["total_tokens"].as_u64().unwrap_or(0) as f64;
            let estimated_cost = total_tokens * 1.25 / 1000000.0;
            println!("   Estimated cost: ${:.6}", estimated_cost);
        }
        
        println!("\n✅ REAL Analysis of YOUR Code Complete!");
        println!("======================================");
        
    } else {
        let error_text = response.text().await?;
        println!("❌ API call failed: {}", error_text);
    }
    
    Ok(())
}
