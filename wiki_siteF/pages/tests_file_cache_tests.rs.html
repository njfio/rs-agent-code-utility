    Overview

    This module contains unit tests for the FileCache implementation located in src/file_cache.rs. The tests verify correctness of cache creation, insertion, retrieval, eviction policies, expiration handling, and thread‑safety. They serve as both regression checks and usage examples for developers integrating the cache into their applications.

    Deep Dive

      In‑memory index backed by on‑disk files.

    Key APIs

    • pub fn new<P: AsRef<Path>>(dir: P, max_size: u64, ttl: Option<Duration>) -> Result<Self, CacheError>
      Creates a cache rooted at `dir` with an optional maximum total size and default TTL for entries.
    • pub fn set<K: AsRef<str>, V: AsRef<[u8]>>(&self, key: K, value: V) -> Result<(), CacheError>
      Writes `value` to a file derived from `key`, updates metadata, and triggers eviction if needed.
    • pub fn get<K: AsRef<str>>(&self, key: K) -> Result<Option<Vec<u8>>, CacheError>
      Returns the cached bytes for `key` if present and not expired; otherwise returns `None`.
    • pub fn remove<K: AsRef<str>>(&self, key: K) -> Result<bool, CacheError>
      Deletes the cached file and metadata for `key`. Returns `true` if an entry was removed.
    • pub fn clear(&self) -> Result<(), CacheError>
      Empties the entire cache directory and resets internal state.
    • pub fn purge_expired(&self) -> Result<usize, CacheError>
      Scans metadata for entries whose TTL has elapsed, removes them, and returns the count of purged items.

    Examples

    Basic cache usage
    use std::time::Duration;
use tempfile::tempdir;
use mycrate::file_cache::FileCache;

fn main() -> Result<(), Box<dyn std::error::Error>> {
    let dir = tempdir()?;
    let cache = FileCache::new(dir.path(), 10 * 1024 * 1024, Some(Duration::from_secs(60)))?;
    cache.set("greeting", b"Hello, world!")?;
    let value = cache.get("greeting")?.expect("value missing");
    assert_eq!(value, b"Hello, world!");
    Ok(())
}

    Creates a temporary cache, stores a byte slice, retrieves it, and asserts the round‑trip works. The example demonstrates error handling with ? and the optional TTL parameter.

    Concurrent access pattern
    use std::sync::Arc;
use std::thread;
use mycrate::file_cache::FileCache;

fn main() {
    let cache = Arc::new(FileCache::new("/tmp/mycache", 50_000_000, None).unwrap());
    let mut handles = Vec::new();
    for i in 0..8 {
        let c = Arc::clone(&cache);
        handles.push(thread::spawn(move || {
            let key = format!("key-{}", i);
            c.set(&key, b"data").unwrap();
            let _ = c.get(&key).unwrap();
        }));
    }
    for h in handles { h.join().unwrap(); }
}

    Spawns several threads that each write and read distinct keys. The Arc wrapper together with the internal RwLock guarantees safe concurrent access.

    Security

    Risks
    • Path traversal if keys are not sanitized – a crafted key could escape the cache directory.
    • Unbounded cache size may lead to disk exhaustion on the host system.
    • Storing sensitive data unencrypted on disk could be read by other users with filesystem access.
    Mitigations
    • Keys are hashed and stored under a fixed subdirectory; never concatenate raw keys into paths.
    • Enforce `max_size` and optionally a per‑file size limit; tests verify eviction logic.
    • Provide an optional encryption layer or store only non‑secret data; document that `FileCache` does not perform encryption.

      fn test_read_to_string_and_caching() -> Result<(), Box<dyn std::error::Error>> {
    let cache = FileCache::new();
    let temp_dir = TempDir::new()?;
    let test_file = temp_dir.path().join("test.txt");
    
    let content = "Hello, world!";
    fs::write(&test_file, content)?;
    
    // File should not be cached initially
    assert!(!cache.contains(&test_file));
    assert_eq!(cache.len(), 0);
    
    // Read file content (should cache it)
    let read_content = cache.read_to_string(&test_file)?;
    assert_eq!(read_content, content);
    
    // File should now be cached
    assert!(cache.contains(&test_file));
    assert_eq!(cache.len(), 1);
    
    // Read again (should hit cache)
    let read_content2 = cache.read_to_string(&test_file)?;
    assert_eq!(read_content2, content);
    
    Ok(())
}

    assert_eq!(stats_after_hit.cached_files, 1);
    
    Ok(())

      fn test_hit_ratio() -> Result<(), Box<dyn std::error::Error>> {
    let cache = FileCache::new();
    let temp_dir = TempDir::new()?;
    let test_file = temp_dir.path().join("test.txt");

    // Read file again (hit)
    cache.read_to_string(&test_file)?;

    cache.read_to_string(&test_file)?;
    assert!((cache.hit_ratio() - 0.6667).abs() < 0.001); // 2 hits, 1 miss ≈ 66.67%
    
    Ok(())

      fn test_multiple_files() -> Result<(), Box<dyn std::error::Error>> {
    let cache = FileCache::new();
    let temp_dir = TempDir::new()?;
    
    let file1 = temp_dir.path().join("file1.txt");
    let file2 = temp_dir.path().join("file2.txt");
    let file3 = temp_dir.path().join("file3.txt");
    
    fs::write(&file1, "Content 1")?;
    fs::write(&file2, "Content 2")?;
    fs::write(&file3, "Content 3")?;
    
    // Cache all files
    let content1 = cache.read_to_string(&file1)?;
    let content2 = cache.read_to_string(&file2)?;
    let content3 = cache.read_to_string(&file3)?;
    
    assert_eq!(content1, "Content 1");
    assert_eq!(content2, "Content 2");
    assert_eq!(content3, "Content 3");
    
    // All files should be cached
    assert_eq!(cache.len(), 3);
    assert!(cache.contains(&file1));
    assert!(cache.contains(&file2));
    assert!(cache.contains(&file3));
    
    // Stats should show 3 misses, 0 hits
    let stats = cache.stats();
    assert_eq!(stats.hits, 0);
    assert_eq!(stats.misses, 3);
    assert_eq!(stats.cached_files, 3);
    
    Ok(())
}

      fn test_average_file_size() -> Result<(), Box<dyn std::error::Error>> {
    let cache = FileCache::new();
    let temp_dir = TempDir::new()?;
    
    let file1 = temp_dir.path().join("small.txt");
    let file2 = temp_dir.path().join("large.txt");
    
    fs::write(&file1, "Hi")?; // 2 bytes
    fs::write(&file2, "Hello, world!")?; // 13 bytes
    
    // Cache both files
    cache.read_to_string(&file1)?;
    cache.read_to_string(&file2)?;
    
    let stats = cache.stats();
    let avg_size = stats.average_file_size();
    
    // Average should be (2 + 13) / 2 = 7.5 bytes
    assert!((avg_size - 7.5).abs() < 0.1);
    
    Ok(())
}
