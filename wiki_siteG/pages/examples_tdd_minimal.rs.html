    Overview

    The examples/tdd_minimal.rs file demonstrates a minimal Test‑Driven Development (TDD) workflow in Rust. It defines a simple arithmetic library with an add function and a Calculator struct, accompanied by unit tests that illustrate how to write, run, and iterate tests using Rust's built‑in test framework.

    Deep Dive

    • function add
      Adds two `i32` values and returns the sum.
    • struct Calculator
      A lightweight calculator exposing basic operations.
    • module tests
      Unit tests exercising `add` and `Calculator::multiply`.

    Key APIs

    • pub fn add(a: i32, b: i32) -> i32
      let sum = add(10, 15); // sum == 25
    • pub fn new() -> Self
      let calc = Calculator::new();
    • pub fn multiply(&self, a: i32, b: i32) -> i32
      let product = calc.multiply(6, 7); // product == 42
