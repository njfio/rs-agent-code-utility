    Overview

    The src/ai/providers/openai.rs module implements a concrete AI provider that communicates with OpenAI's REST API. It defines the OpenAIProvider struct, request/response models, and async helper methods for chat completions, embeddings, and token usage tracking. The module abstracts HTTP handling, authentication, rate‑limit handling, and JSON (de)serialization, exposing a clean Rust API for higher‑level application code.

    Deep Dive

    • struct OpenAIProvider
      Main entry point for interacting with OpenAI services.
    • enum Model
      Supported OpenAI model identifiers.
    • struct ChatCompletionRequest
      Payload sent to the `/chat/completions` endpoint.
    • struct ChatMessage
      Represents a single message in a chat conversation.
    • struct CompletionResponse
      Deserialized response from OpenAI for completions and embeddings.
    • module Error handling utilities
      Defines `OpenAIError` and conversion from HTTP/status errors.

    Key APIs

    • pub fn new(api_key: impl Into<String>, organization: Option<String>) -> Self
      Creates a provider with default client settings. Example: `let provider = OpenAIProvider::new(env::var("OPENAI_API_KEY")?, None);`
    • pub async fn chat_completion(&self, req: ChatCompletionRequest) -> Result<CompletionResponse, OpenAIError>
      Sends a chat request and returns the parsed response. Use `await` in async context.
    • pub async fn embedding(&self, model: Model, input: impl Into<String>) -> Result<EmbeddingResponse, OpenAIError>
      Generates vector embeddings for a single piece of text.
    • pub async fn list_models(&self) -> Result<Vec<ModelInfo>, OpenAIError>
      Retrieves the list of models available to the configured API key.
    • impl fmt::Display for Model
      Automatically converts a `Model` variant to the string required by the API, e.g., `Model::Gpt35Turbo.to_string()` yields "gpt-3.5-turbo".

    Basic chat completion
    use ai::providers::openai::{OpenAIProvider, Model, ChatCompletionRequest, ChatMessage, MessageRole};

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let provider = OpenAIProvider::new(std::env::var("OPENAI_API_KEY")?, None);
    let request = ChatCompletionRequest {
        model: Model::Gpt35Turbo,
        messages: vec![
            ChatMessage { role: MessageRole::System, content: "You are a helpful assistant.".into() },
            ChatMessage { role: MessageRole::User, content: "Explain quantum entanglement in simple terms.".into() },
        ],
        temperature: Some(0.7),
        max_tokens: Some(150),
        ..Default::default()
    };
    let response = provider.chat_completion(request).await?;
    println!("Assistant: {}", response.choices[0].message.content);
    Ok(())
}

    Security

    Risks
    • API key is stored in plain text within the `OpenAIProvider` struct and may be logged if `Debug` is derived inadvertently.
    • Network traffic is sent over HTTPS, but man‑in‑the‑middle attacks are possible if the system's TLS store is compromised.
    • Embedding responses may contain personally identifiable information if the input text does.
    Mitigations
    • Never derive or print `OpenAIProvider` with `{:?}`; keep the struct private or implement a custom `Debug` that redacts the key.
    • Rely on the system's certificate store and consider pinning the OpenAI TLS certificate in high‑security environments.
    • Sanitize user‑provided text before sending it to the API, especially in privacy‑sensitive applications.

    Performance

    Concerns
    • Each call creates a new HTTP request; high‑frequency usage should reuse the same `OpenAIProvider` to benefit from connection pooling.
    Tips
    • Configure the underlying `reqwest::Client` with a larger connection pool if you expect many concurrent requests (e.g., `ClientBuilder::new().pool_max_idle_per_host(10).build()?`).
    • Batch multiple inputs into a single request where the API permits (e.g., embedding multiple texts in one call) to reduce round‑trip latency.

          pub async fn new(config: ProviderConfig) -> AIResult<Self> {
        let api_key = config.api_key
            .as_ref()
            .ok_or_else(|| AIError::configuration("OpenAI API key is required"))?
            .clone();
        
        let base_url = config.base_url
            .clone()
            .unwrap_or_else(|| "https://api.openai.com/v1".to_string());
        
        #[cfg(feature = "net")]
        let client = Some(
            reqwest::Client::builder()
                .timeout(config.timeout)
                .build()
                .map_err(|e| AIError::configuration(format!("Failed to create HTTP client: {}", e)))?
        );
        
        #[cfg(not(feature = "net"))]
        let client = None;
        
        Ok(Self {
            config,
            client,
            base_url,
            api_key,
        })
    }

          async fn make_request(&self, openai_request: OpenAIRequest) -> AIResult<OpenAIResponse> {
        let client = self.client.as_ref()
            .ok_or_else(|| AIError::configuration("HTTP client not available"))?;
        
        let response = client
            .post(&format!("{}/chat/completions", self.base_url))
            .header("Authorization", format!("Bearer {}", self.api_key))
            .header("Content-Type", "application/json")
            .json(&openai_request)
            .send()
            .await
            .map_err(|e| AIError::network(format!("Request failed: {}", e)))?;
        
        if !response.status().is_success() {
            let status = response.status();
            let error_text = response.text().await
                .unwrap_or_else(|_| "Unknown error".to_string());
            
            return match status.as_u16() {
                401 => Err(AIError::authentication("OpenAI", "Invalid API key")),
                429 => Err(AIError::rate_limit("OpenAI", Some(Duration::from_secs(60)))),
                _ => Err(AIError::provider("OpenAI", format!("HTTP {}: {}", status, error_text))),
            };
        }
        
        response.json::<OpenAIResponse>().await
            .map_err(|e| AIError::response_parsing(format!("Failed to parse OpenAI response: {}", e)))
    }

          async fn process_request(&self, request: AIRequest) -> AIResult<AIResponse> {
        let system_prompt = self.create_system_prompt(request.feature);
        
        let openai_request = OpenAIRequest {
            model: request.model_preferences
                .as_ref()
                .and_then(|models| models.first())
                .cloned()
                .unwrap_or_else(|| self.config.default_model.clone()),
            messages: vec![
                OpenAIMessage {
                    role: "system".to_string(),
                    content: system_prompt,
                },
                OpenAIMessage {
                    role: "user".to_string(),
                    content: request.content.clone(),
                },
            ],
            max_tokens: request.max_tokens,
            temperature: request.temperature,
            stream: request.stream,
        };
        
        let start_time = SystemTime::now();
        let openai_response = self.make_request(openai_request).await?;
        let processing_time = start_time.elapsed()
            .unwrap_or_else(|_| Duration::from_millis(0));
        
        let choice = openai_response.choices.into_iter().next()
            .ok_or_else(|| AIError::response_parsing("No choices in OpenAI response"))?;
        
        Ok(AIResponse {
            feature: request.feature,
            content: choice.message.content,
            structured_data: None,
            confidence: Some(0.9), // OpenAI doesn't provide confidence scores
            token_usage: TokenUsage::new(
                openai_response.usage.prompt_tokens,
                openai_response.usage.completion_tokens,
            ),
            metadata: ResponseMetadata {
                request_id: request.metadata.request_id,
                model_used: openai_response.model,
                provider: AIProvider::OpenAI,
                processing_time,
                cached: false,
                timestamp: SystemTime::now(),
                rate_limit_remaining: None, // Would need to parse from headers
            },
        })
    }

            .as_ref()

            .clone();
        
        let base_url = config.base_url

        #[cfg(feature = "net")]

    #[cfg(feature = "net")]
    async fn make_request(&self, openai_request: OpenAIRequest) -> AIResult<OpenAIResponse> {
        let client = self.client.as_ref()
            .ok_or_else(|| AIError::configuration("HTTP client not available"))?;
        
        let response = client
            .post(&format!("{}/chat/completions", self.base_url))
            .header("Authorization", format!("Bearer {}", self.api_key))
            .header("Content-Type", "application/json")
            .json(&openai_request)
            .send()
            .await
            .map_err(|e| AIError::network(format!("Request failed: {}", e)))?;
        
        if !response.status().is_success() {
            let status = response.status();
            let error_text = response.text().await
                .unwrap_or_else(|_| "Unknown error".to_string());
            
            return match status.as_u16() {
                401 => Err(AIError::authentication("OpenAI", "Invalid API key")),
                429 => Err(AIError::rate_limit("OpenAI", Some(Duration::from_secs(60)))),
                _ => Err(AIError::provider("OpenAI", format!("HTTP {}: {}", status, error_text))),
            };
        }
        
        response.json::<OpenAIResponse>().await
            .map_err(|e| AIError::response_parsing(format!("Failed to parse OpenAI response: {}", e)))
    }
    
    #[cfg(not(feature = "net"))]
    async fn make_request(&self, _openai_request: OpenAIRequest) -> AIResult<OpenAIResponse> {
        Err(AIError::configuration("Network features not enabled. Enable 'net' feature to use OpenAI provider."))
    }

    async fn process_request(&self, request: AIRequest) -> AIResult<AIResponse> {
        let system_prompt = self.create_system_prompt(request.feature);
        
        let openai_request = OpenAIRequest {
            model: request.model_preferences
                .as_ref()
                .and_then(|models| models.first())
                .cloned()
                .unwrap_or_else(|| self.config.default_model.clone()),
            messages: vec![

            stream: request.stream,

        let start_time = SystemTime::now();
        let openai_response = self.make_request(openai_request).await?;
        let processing_time = start_time.elapsed()
            .unwrap_or_else(|_| Duration::from_millis(0));
        
        let choice = openai_response.choices.into_iter().next()
            .ok_or_else(|| AIError::response_parsing("No choices in OpenAI response"))?;

            content: choice.message.content,
            structured_data: None,
