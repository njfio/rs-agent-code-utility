    Overview

    The src/ai/providers/openai.rs module implements a concrete provider for interacting with OpenAI's REST API. It offers high‑level, async‑ready functions for text completions, chat completions, embeddings, and fine‑tuned model management. The provider abstracts HTTP details behind a thin, ergonomic Rust API, handling authentication, request/response (de)serialization, pagination, streaming, and error mapping.

    Key responsibilities:

    • Manage the OpenAI API key and endpoint configuration.
    • Serialize request structs (CompletionRequest, ChatRequest, EmbeddingRequest, …) into JSON.
    • Perform HTTP calls using reqwest (or a pluggable client) and deserialize responses into typed structs.
    • Expose both blocking and streaming interfaces.
    • Provide helper utilities for token counting, model selection, and rate‑limit handling.

    The module is deliberately kept provider‑specific; generic client logic lives in src/ai/client.rs and model‑agnostic traits are defined in src/ai/providers/mod.rs.

    Deep Dive

    • struct OpenAIProvider
      Main entry point for all OpenAI API interactions.
    • enum Model
      Enum of supported OpenAI model identifiers.
    • struct CompletionRequest
      Parameters for a text completion request.
    • struct CompletionResponse
      Deserialized response for a completion request.
    • struct ChatRequest
      Parameters for a chat‑completion request.
    • struct EmbeddingRequest
      Parameters for an embedding generation request.
    • struct ErrorResponse
      Standard OpenAI error payload.
    • enum StreamChunk
      Represents a single chunk of a streaming response.

    Key APIs

    • pub fn new(api_key: impl Into<String>) -> Result<Self, ProviderError>
      Creates a provider instance. The function validates the key format and builds an internal `reqwest::Client` with the appropriate `Authorization` header.
    • pub async fn completion(&self, req: CompletionRequest) -> Result<CompletionResponse, ProviderError>
      Sends a one‑shot completion request. Returns the fully materialized response; use `req.stream = true` and `completion_stream` for streaming.
    • pub fn completion_stream(&self, req: CompletionRequest) -> impl Stream<Item = Result<StreamChunk, ProviderError>>
      Provides a lazy async stream of `StreamChunk`s for real‑time token generation. Typical usage with `while let Some(chunk) = stream.next().await`.
    • pub async fn chat_completion(&self, req: ChatRequest) -> Result<ChatResponse, ProviderError>
      Convenient wrapper for the `/v1/chat/completions` endpoint. Handles message ordering and optional streaming via `chat_stream`.
    • pub async fn embedding(&self, req: EmbeddingRequest) -> Result<EmbeddingResponse, ProviderError>
      Generates vector embeddings for a batch of input strings. Returns a struct containing the embedding vectors and usage statistics.
    • impl Display for Model
      Converts a `Model` enum variant into the exact model identifier string required by the API (e.g., `gpt-4-turbo`).

    Examples

    Basic text completion
    use ai::providers::openai::{OpenAIProvider, CompletionRequest, Model};

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    // Initialise the provider – the API key is read from the OPENAI_API_KEY env var.
    let provider = OpenAIProvider::new(std::env::var("OPENAI_API_KEY")?)?;

    let request = CompletionRequest {
        model: Model::Gpt35Turbo,
        prompt: "Explain the theory of relativity in one sentence.".into(),
        max_tokens: Some(60),
        temperature: Some(0.7),
        ..Default::default()
    };

    let response = provider.completion(request).await?;
    println!("AI says: {}", response.first_text());
    Ok(())
}

          async fn make_request(&self, openai_request: OpenAIRequest) -> AIResult<OpenAIResponse> {
        let client = self.client.as_ref()
            .ok_or_else(|| AIError::configuration("HTTP client not available"))?;
        
        let response = client
            .post(&format!("{}/chat/completions", self.base_url))
            .header("Authorization", format!("Bearer {}", self.api_key))
            .header("Content-Type", "application/json")
            .json(&openai_request)
            .send()
            .await
            .map_err(|e| AIError::network(format!("Request failed: {}", e)))?;
        
        if !response.status().is_success() {
            let status = response.status();
            let error_text = response.text().await
                .unwrap_or_else(|_| "Unknown error".to_string());
            
            return match status.as_u16() {
                401 => Err(AIError::authentication("OpenAI", "Invalid API key")),
                429 => Err(AIError::rate_limit("OpenAI", Some(Duration::from_secs(60)))),
                _ => Err(AIError::provider("OpenAI", format!("HTTP {}: {}", status, error_text))),
            };
        }
        
        response.json::<OpenAIResponse>().await
            .map_err(|e| AIError::response_parsing(format!("Failed to parse OpenAI response: {}", e)))
    }

          async fn process_request(&self, request: AIRequest) -> AIResult<AIResponse> {
        let system_prompt = self.create_system_prompt(request.feature);
        
        let openai_request = OpenAIRequest {
            model: request.model_preferences
                .as_ref()
                .and_then(|models| models.first())
                .cloned()
                .unwrap_or_else(|| self.config.default_model.clone()),
            messages: vec![
                OpenAIMessage {
                    role: "system".to_string(),
                    content: system_prompt,
                },
                OpenAIMessage {
                    role: "user".to_string(),
                    content: request.content.clone(),
                },
            ],
            max_tokens: request.max_tokens,
            temperature: request.temperature,
            stream: request.stream,
        };
        
        let start_time = SystemTime::now();
        let openai_response = self.make_request(openai_request).await?;
        let processing_time = start_time.elapsed()
            .unwrap_or_else(|_| Duration::from_millis(0));
        
        let choice = openai_response.choices.into_iter().next()
            .ok_or_else(|| AIError::response_parsing("No choices in OpenAI response"))?;
        
        Ok(AIResponse {
            feature: request.feature,
            content: choice.message.content,
            structured_data: None,
            confidence: Some(0.9), // OpenAI doesn't provide confidence scores
            token_usage: TokenUsage::new(
                openai_response.usage.prompt_tokens,
                openai_response.usage.completion_tokens,
            ),
            metadata: ResponseMetadata {
                request_id: request.metadata.request_id,
                model_used: openai_response.model,
                provider: AIProvider::OpenAI,
                processing_time,
                cached: false,
                timestamp: SystemTime::now(),
                rate_limit_remaining: None, // Would need to parse from headers
            },
        })
    }

    #[cfg(feature = "net")]
    async fn make_request(&self, openai_request: OpenAIRequest) -> AIResult<OpenAIResponse> {
        let client = self.client.as_ref()
            .ok_or_else(|| AIError::configuration("HTTP client not available"))?;
        
        let response = client
            .post(&format!("{}/chat/completions", self.base_url))
            .header("Authorization", format!("Bearer {}", self.api_key))
            .header("Content-Type", "application/json")
            .json(&openai_request)
            .send()
            .await
            .map_err(|e| AIError::network(format!("Request failed: {}", e)))?;
        
        if !response.status().is_success() {
            let status = response.status();
            let error_text = response.text().await
                .unwrap_or_else(|_| "Unknown error".to_string());
            
            return match status.as_u16() {
                401 => Err(AIError::authentication("OpenAI", "Invalid API key")),
                429 => Err(AIError::rate_limit("OpenAI", Some(Duration::from_secs(60)))),
                _ => Err(AIError::provider("OpenAI", format!("HTTP {}: {}", status, error_text))),
            };
        }
        
        response.json::<OpenAIResponse>().await
            .map_err(|e| AIError::response_parsing(format!("Failed to parse OpenAI response: {}", e)))
    }

        Err(AIError::configuration("Network features not enabled. Enable 'net' feature to use OpenAI provider."))
    }
