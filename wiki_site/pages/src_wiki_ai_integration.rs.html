    AI Commentary

    Overview

    The src/wiki/ai_integration.rs module provides a high‑level, ergonomic wrapper around external AI services (e.g., OpenAI, Anthropic). It abstracts HTTP communication, authentication, model selection, and response parsing, allowing the rest of the codebase to interact with AI models via a simple, type‑safe API. The module supports both synchronous and asynchronous usage, streaming responses, and configurable retry/back‑off policies.

    Deep Dive

    • struct AiClient
      Core client that holds configuration, authentication, and an HTTP transport.
    • enum Model
      Supported AI models and their identifiers.
    • struct Prompt
      Encapsulates a user prompt and optional system messages.
    • enum Message
      Single message in a chat‑style interaction.
    • enum AiError
      Error type returned by the module.
    • fn send_prompt
      Sends a prompt to the configured model and returns the full response.
    • fn stream_prompt
      Streams a response token‑by‑token using Server‑Sent Events or HTTP chunking.

    Key APIs

    • pub fn new(api_key: impl Into<String>, model: Model) -> Self
      Creates a client with default HTTP settings. Example: `let client = AiClient::new(env::var("OPENAI_KEY")?, Model::Gpt35Turbo);`
    • pub fn set_api_key(&mut self, key: impl Into<String>)
      Updates the API key at runtime, useful for multi‑tenant scenarios.
    • pub async fn send_prompt(&self, prompt: Prompt) -> Result<String, AiError>
      Sends a complete prompt and awaits the final answer. Ideal for batch jobs.
    • pub async fn stream_prompt(&self, prompt: Prompt) -> impl Stream<Item = Result<String, AiError>>
      Returns an async stream of partial responses for real‑time UI rendering.

    Examples

    Basic synchronous usage (tokio runtime)
    use wiki::ai_integration::{AiClient, Model, Prompt, Message};

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = AiClient::new(std::env::var("OPENAI_KEY")?, Model::Gpt35Turbo);
    let prompt = Prompt {
        system: Some("You are a helpful assistant.".into()),
        messages: vec![Message::User("Explain quantum entanglement in simple terms.".into())],
    };
    let answer = client.send_prompt(prompt).await?;
    println!("AI says: {}", answer);
    Ok(())
}

    Creates a client, builds a chat prompt, sends it, and prints the response. The example uses Tokio because the API is async.

    Streaming response for a CLI chat
    use wiki::ai_integration::{AiClient, Model, Prompt, Message};
use futures_util::StreamExt;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = AiClient::new(std::env::var("OPENAI_KEY")?, Model::Gpt4);
    let prompt = Prompt {
        system: None,
        messages: vec![Message::User("Write a haiku about Rust.".into())],
    };
    let mut stream = client.stream_prompt(prompt).await?;
    while let Some(chunk) = stream.next().await {
        print!("{}", chunk?);
    }
    Ok(())
}

    Demonstrates how to consume the async stream returned by stream_prompt, printing each token as it arrives.

    use wiki::ai_integration::{AiClient, Model, Prompt, Message};

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let mut client = AiClient::new(std::env::var("OPENAI_KEY")?, Model::Gpt35Turbo);

    Security

    Risks
    • API key exposure in logs or version control.
    • Sending user‑generated content to third‑party models may violate privacy policies.
    • Potential for prompt injection attacks where malicious input manipulates model behavior.
    Mitigations
    • Store keys in environment variables or secret vaults; never log the raw key.
    • Sanitize or redact personally identifiable information before sending it to the AI service.
    • Implement a whitelist of allowed prompt patterns and enforce length limits.

          pub(super) fn parse_ai_json<T: for<'de> serde::Deserialize<'de>>(raw: &str) -> Option<T> {
        let s = raw.trim();
        let s = s
            .strip_prefix("```json")
            .and_then(|x| x.strip_suffix("```"))
            .map(|x| x.trim())
            .unwrap_or(s);
        serde_json::from_str::<T>(s).ok()
    }

          pub(super) fn make_ai_builder(&self) -> crate::ai::service::AIServiceBuilder {
        use crate::ai::service::AIServiceBuilder;
        use crate::ai::config::{ProviderConfig, ModelConfig, RateLimitConfig, RetryConfig};

        let mut builder = AIServiceBuilder::new();
        let use_groq = std::env::var("AI_GROQ_API_KEY").is_ok() || std::env::var("GROQ_API_KEY").is_ok();
        if use_groq {
            let api_key = std::env::var("AI_GROQ_API_KEY").or_else(|_| std::env::var("GROQ_API_KEY")).ok();
            let groq_cfg = ProviderConfig {
                enabled: true,
                api_key,
                base_url: Some("https://api.groq.com/openai/v1".to_string()),
                organization: None,
                models: vec![ModelConfig {
                    name: "openai/gpt-oss-120b".to_string(),
                    context_length: 8192,
                    max_tokens: 8001,
                    supports_streaming: false,
                    cost_per_token: None,
                    supported_features: vec![
                        crate::ai::types::AIFeature::DocumentationGeneration,
                        crate::ai::types::AIFeature::CodeExplanation,
                        crate::ai::types::AIFeature::SecurityAnalysis,
                        crate::ai::types::AIFeature::RefactoringSuggestions,
                        crate::ai::types::AIFeature::QualityAssessment,
                    ],
                }],
                default_model: "openai/gpt-oss-120b".to_string(),
                timeout: std::time::Duration::from_secs(30),
                rate_limit: RateLimitConfig::default(),
                retry: RetryConfig::default(),
            };
            builder = builder
                .with_provider(crate::ai::types::AIProvider::Groq, groq_cfg)
                .with_default_provider(crate::ai::types::AIProvider::Groq);
        } else {
            builder = builder.with_default_provider(crate::ai::types::AIProvider::OpenAI);
        }
        let use_groq = std::env::var("AI_GROQ_API_KEY").is_ok() || std::env::var("GROQ_API_KEY").is_ok();
        let builder = if self.config.ai_use_mock
            || (!use_groq
                && std::env::var("OPENAI_API_KEY").is_err()
                && std::env::var("AI_OPENAI_API_KEY").is_err())
        {
            builder.with_mock_providers(true)
        } else {
            builder
        };
        builder
    }

        // Minimal HTML rendering for AI JSON doc
        let mut out = String::new();
        use std::fmt::Write as _;
        let _ = writeln!(&mut out, "<div class=\"card ai\"><h3>AI Commentary</h3>");
        if let Some(ov) = &doc.overview {
            let _ = writeln!(&mut out, "<h4>Overview</h4>{}", super::util::markdown_to_html(ov));
        }
        if let Some(items) = &doc.deep_dive { if !items.is_empty() { let _ = writeln!(&mut out, "<h4>Deep Dive</h4><ul>");
            for i in items {
                let name = i.name.as_deref().unwrap_or("");
                let kind = i.kind.as_deref().unwrap_or("");
                let sum = i.summary.as_deref().unwrap_or("");
                if !name.is_empty() {
                    let anchor = format!("#symbol-{}", super::util::anchorize(name));
                    let _ = writeln!(&mut out, "<li><strong>{}</strong> <a href=\"{}\">{}</a><br>{}</li>", super::util::html_escape(kind), super::util::html_escape(&anchor), super::util::html_escape(name), super::util::html_escape(sum));
                } else {
                    let _ = writeln!(&mut out, "<li><strong>{}</strong><br>{}</li>", super::util::html_escape(kind), super::util::html_escape(sum));
                }
            }
            let _ = writeln!(&mut out, "</ul>"); }}
        if let Some(apis) = &doc.key_apis { if !apis.is_empty() { let _ = writeln!(&mut out, "<h4>Key APIs</h4><ul>");
            for a in apis {
                let sig = a.signature.as_deref().unwrap_or("");
                let useg = a.usage.as_deref().unwrap_or("");
                let _ = writeln!(&mut out, "<li><code>{}</code><br><small>{}</small></li>", super::util::html_escape(sig), super::util::html_escape(useg));
            }
            let _ = writeln!(&mut out, "</ul>"); }}
        if let Some(exs) = &doc.examples { if !exs.is_empty() { let _ = writeln!(&mut out, "<h4>Examples</h4>");

                let expl = e.explanation.as_deref().unwrap_or("");
                let title = e.title.as_deref().unwrap_or("");
                if !title.is_empty() { let _ = writeln!(&mut out, "<h5>{}</h5>", super::util::html_escape(title)); }
                let _ = writeln!(&mut out, "<div class=\"card\"><pre><code class=\"lang-{}\">{}</code></pre>{}</div>", super::util::html_escape(lang), super::util::html_escape(code), super::util::markdown_to_html(expl));
            }
        }}
        if let Some(sec) = &doc.security { if sec.risks.as_ref().map(|v| !v.is_empty()).unwrap_or(false) || sec.mitigations.as_ref().map(|v| !v.is_empty()).unwrap_or(false) {
            let _ = writeln!(&mut out, "<h4>Security</h4>");

    /// Parse JSON (or ```json fenced) into a target type
    pub(super) fn parse_ai_json<T: for<'de> serde::Deserialize<'de>>(raw: &str) -> Option<T> {
        let s = raw.trim();
        let s = s
            .strip_prefix("```json")
            .and_then(|x| x.strip_suffix("```"))
            .map(|x| x.trim())
            .unwrap_or(s);
        serde_json::from_str::<T>(s).ok()
    }

    /// Create an AI builder configured from env + config (no runtime side-effects)
    pub(super) fn make_ai_builder(&self) -> crate::ai::service::AIServiceBuilder {
        use crate::ai::service::AIServiceBuilder;
        use crate::ai::config::{ProviderConfig, ModelConfig, RateLimitConfig, RetryConfig};

        let mut builder = AIServiceBuilder::new();
        let use_groq = std::env::var("AI_GROQ_API_KEY").is_ok() || std::env::var("GROQ_API_KEY").is_ok();
        if use_groq {
            let api_key = std::env::var("AI_GROQ_API_KEY").or_else(|_| std::env::var("GROQ_API_KEY")).ok();
            let groq_cfg = ProviderConfig {
                enabled: true,
                api_key,
                base_url: Some("https://api.groq.com/openai/v1".to_string()),
                organization: None,
                models: vec![ModelConfig {
                    name: "openai/gpt-oss-120b".to_string(),
                    context_length: 8192,
                    max_tokens: 8001,
                    supports_streaming: false,
                    cost_per_token: None,
                    supported_features: vec![
                        crate::ai::types::AIFeature::DocumentationGeneration,
                        crate::ai::types::AIFeature::CodeExplanation,
                        crate::ai::types::AIFeature::SecurityAnalysis,
                        crate::ai::types::AIFeature::RefactoringSuggestions,
                        crate::ai::types::AIFeature::QualityAssessment,
                    ],
                }],
                default_model: "openai/gpt-oss-120b".to_string(),
                timeout: std::time::Duration::from_secs(30),
                rate_limit: RateLimitConfig::default(),
                retry: RetryConfig::default(),
            };
            builder = builder
                .with_provider(crate::ai::types::AIProvider::Groq, groq_cfg)
                .with_default_provider(crate::ai::types::AIProvider::Groq);
        } else {
            builder = builder.with_default_provider(crate::ai::types::AIProvider::OpenAI);
        }
        let use_groq = std::env::var("AI_GROQ_API_KEY").is_ok() || std::env::var("GROQ_API_KEY").is_ok();
