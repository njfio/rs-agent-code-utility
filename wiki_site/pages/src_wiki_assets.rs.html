    Overview

    The src/wiki/assets.rs module provides a comprehensive, type‑safe system for managing static assets used by the wiki engine – images, markdown files, JSON data, and binary blobs. It abstracts file‑system access, caching, versioning and serialization behind a small set of ergonomic APIs, allowing the rest of the codebase to request assets by logical identifiers without worrying about paths or I/O details.

    Deep Dive

    • struct AssetId
      Unique logical identifier for an asset, typically a kebab‑case string like `"images/logo.png"`.
    • enum Asset
      Represents the concrete data of a loaded asset.
    • enum AssetError
      Error type returned by all asset‑related operations.
    • struct AssetStore
      Central repository that caches loaded assets and resolves `AssetId` to concrete data.
    • fn load_asset
      Synchronously load an asset from disk, deserialising it into the appropriate `Asset` variant.
    • fn save_asset
      Persist an `Asset` back to its source file, creating parent directories as needed.

    Key APIs

    • pub fn new(base_path: impl AsRef<Path>) -> Self
      Creates a fresh store with the given root directory. All asset look‑ups are resolved relative to this path.
    • pub fn load(&self, id: impl Into<AssetId>) -> Result<Arc<Asset>, AssetError>
      High‑level entry point used throughout the wiki code. It returns a shared reference to the cached asset or loads it on‑demand.
    • pub fn save(&self, id: impl Into<AssetId>, asset: &Asset) -> Result<(), AssetError>
      Writes a modified asset back to disk, updating the cache atomically.
    • pub fn register_loader(&self, ext: &str, loader: Box<dyn AssetLoader>)
      Allows custom deserialisers (e.g., TOML, custom binary formats) to be plugged in for new file extensions.
    • pub fn as_text(&self) -> Option<&str>
      Conveniently extracts a UTF‑8 string from a `Text` variant; returns `None` for other variants.

    fn load(&self, path: &Path) -> Result<Asset, AssetError> {
        let data = std::fs::read_to_string(path)?;
        let value: toml::Value = toml::from_str(&data).map_err(|e| AssetError::Deserialize(e.to_string()))?;
        Ok(Asset::Json(serde_json::to_value(value).unwrap()))
    }
}

    pub(super) fn postprocess_cdn_refs_impl(&self, out: &Path) -> Result<()> {
        fn process_dir(dir: &Path) -> std::io::Result<()> {
            for entry in std::fs::read_dir(dir)? {
                let entry = entry?;
                let path = entry.path();
                if path.is_dir() { process_dir(&path)?; continue; }
                if let Some(ext) = path.extension() { if ext == "html" {
                    if let Ok(content) = std::fs::read_to_string(&path) {
                        let mut replaced = content.replace("assets/mermaid.js", "assets/mermaid.js");
                        replaced = replaced.replace("assets/mermaid.js", "assets/mermaid.js");
                        replaced = replaced.replace("assets/hljs.js", "assets/hljs.js");
                        replaced = replaced.replace("assets/hljs.css", "assets/hljs.css");
                        replaced = replaced.replace("assets/", "assets/");
                        if replaced != content { let _ = std::fs::write(&path, replaced); }
                    }
                }}
            }
            Ok(())
        }
        process_dir(out).map_err(|e| e.into())
    }
